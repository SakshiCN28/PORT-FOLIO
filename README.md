# Sakshi Nagonde | AI & ML Developer Portfolio

Welcome to my portfolio! This repository showcases my journey, projects, and skills at the intersection of **Artificial Intelligence, Machine Learning, and Full-Stack Software Development**. 

The portfolio features a sleek, interactive glassmorphic design with a custom particle-based neural network background, real-time interactive playgrounds, and clean responsiveness.

---

## 🚀 Live Demo & Repository
- **GitHub Repository**: [https://github.com/22sakshinagonde/Port-folio.git](https://github.com/22sakshinagonde/Port-folio.git)

## 🛠️ Features

- **Dynamic Interactive UI**: Built with React, Vite, and Tailwind CSS for rapid loading and responsiveness.
- **Neural Network Background**: A custom interactive canvas background (`NeuralBackground.jsx`) that responds to mouse movements, symbolizing neural connections.
- **Project Showcases**: Spotlights various AI/ML and software engineering projects with quick links to code repositories.
- **Interactive Playground**: A dedicated section to demonstrate and interact with features.
- **Contact Pipeline**: A fully styled contact form ready to be integrated with email services.

---

## 💻 Tech Stack

- **Frontend Core**: React 18 (with JSX & Hooks)
- **Styling**: Tailwind CSS (version 3) & PostCSS
- **Build Tool**: Vite (extremely fast development server and optimized production build)
- **Icons**: Lucide React
- **Hosting/Deployment**: Optimized for standard static hosts (like Vercel, Netlify, or GitHub Pages)

## ⚙️ Local Development Setup

To run this portfolio website locally on your computer, follow these simple steps:

### Prerequisites
Make sure you have [Node.js](https://nodejs.org/) installed (v18 or higher recommended).

### Steps
1. **Clone the repository**:
   ```bash
   git clone https://github.com/22sakshinagonde/Port-folio.git
   cd Port-folio
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the development server**:
   ```bash
   npm run dev
   ```
   Open your browser and navigate to `http://localhost:5173`.

4. **Build for production**:
   ```bash
   npm run build
   ```
   This will output optimized, ready-to-deploy static assets into the `dist` directory.
